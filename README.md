# GitHub Pages Login + Signup (HTML/CSS/JS)

## Files
- `index.html`: login page
- `signup.html`: sign up page (redirects to login after creating account)
- `dashboard.html`: simple landing page after login
- `auth.js`: front-end-only auth demo using `localStorage`
- `styles.css`: styling

## Run locally
Open `index.html` in your browser.

## Deploy on GitHub Pages (repo site)
Put these files in your repo root, then on GitHub:
**Settings → Pages → Deploy from a branch → main → / (root)**.

Your site will be at:
`https://YOURUSERNAME.github.io/REPO_NAME/`

## Notes
This is a demo only (no real server, no secure password storage). For real authentication you need a backend.
